常见问题

要开始使用我们的 API，请按照以下步骤操作：
  1. 注册一个账户
  2. 登录到您的账户
  3. 在用户仪表板中获取您的 API 密钥
  4. 选择您想要使用的 API
  5. 按照 API 文档中的说明进行调用

目前，我们的免费 API 服务没有严格的调用次数限制。但是，我们建议用户合理使用，避免频繁、大量的 API 调用，以确保服务的稳定性和其他用户的使用体验。如果您需要大量调用，请联系我们的客户支持。

我们采取了多项措施来保证 API 调用的安全性：
  • 所有的 API 调用都需要使用 HTTPS
  • 每个用户都有唯一的 API 密钥，请妥善保管
  • 我们会定期审查和更新我们的安全措施
  • 如果发现任何安全问题，请立即联系我们

处理 API 调用中的错误时，请注意以下几点：
  • 检查您的 API 密钥是否正确
  • 确保您的请求参数格式正确
  • 查看返回的错误信息，了解具体原因
  • 参考我们的错误代码文档，了解每个错误的含义和解决方法
  • 如果问题持续，请联系我们的技术支持
